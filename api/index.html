    • Salvar um Arquivo

      Control + S Command + S

    • Executar um Arquivo

      Control + R Command + R

    • Interromper um Processo

      Control + C

    • Suspender um Processo

      Control + Z

    • Limpar Terminal

      Control + L

    • Criar um Arquivo

      ~# touch nome_do_arquivo

    • Criar uma Pasta

      ~# mkdir nome_da_pasta

    • Criar uma Estrutura de Pastas

      ~# mkdir -p pasta_a/pasta_b/pasta_c

    • Remover um Arquivo

      ~# rm nome_do_arquivo

    • Remover uma Pasta

      ~# rm -rf nome_da_pasta

    • Mover um Arquivo ou uma Pasta

      ~# mv arquivo_origem arquivo_destino

    • Copiar um Arquivo

      ~# cp arquivo_origem arquivo_destino

    • Copiar uma Pasta

      ~# cp -r pasta_origem pasta_destino
